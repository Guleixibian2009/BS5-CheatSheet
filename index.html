Dropdown Menus

What's a dropdown menu?

A dropdown menu is a kind of menubar which is hidden before a specific button has been clicked and then floats above other elements. It can be a command list or something else, kind like the right-click menu...

Some basic ones

These two dropdown menus are built with buttons and two <ul>s. Bootstrap will translate those two components into whole menus.

Add something else to the dropdown menus

Links and text are boring to you? Try add something more fascinating to your drop down menus. For example, lines, pure link(blue with underline) and even headings!

Dropdown menus don't always need to be a drop"down". It can go up when there is enough space. Also you can add dividers(<hr>s) and headings in the menus.

You can also equip your dropdown menus with icons, search boxes and more.

Popup Modals

Want to pop up something more complex and alerts are no longer enough for you? Then try modal boxes. They are above the whole screen and when modals are activated users can't scroll on the page. You can add buttons and forms in modals. All popup modals must be closed(dismissed) for user experience.

Different kinds of popup modals

Basically a popup modal has a header, a body and a footer just like a card.

5 different kind of modal boxes for U!

Such long code...

Fullscreen when too narrow

Want your modals to be responsive? Then classes modal-fullscreen-*-down are the best for you. They will become fullscreen when the screen's width is narrower than * (e.g. md, sm, xl).

There are five sizes available.

So when you apply those classes to your modal, mobile users will usually recieve fullscreen ones. because phones are always narrower than sm.

Tooltips(with JavaScript)

Want to convey some additional info on a certain piece of text? Then try Tooltips. They appear when users hover on their "parent" elements.

Basic Tooltips

Kinda complex this time because we must init all tooltips by JavaScript manually. Bootstrap won't pre-init them for performance reasons.

Pro Tip: Add type="module" on <script>s if you're using inline script(in case your browser doesn't recognize bootstrap).

Disable tooltips and re-enable them

Only want to show your users tooltips at a specific time or after some events? Then disable those tooltips first and then re-enable them!

These are also achieved by writing JavaScript. Bootstrap provided us a few functions. Use tooltip.toggle(); to show/hide it. Use tooltip.toggleEnabled(); to enable/disable it.

Scrollspy Navbars

Being tired of writing too much JavaScript to build a sidebar? Just use a scrollspy. You just need to add all the headings to a list-group or a <nav>. Scrollspy will handle the scrolling and highlighting.

P.S. You must set the scrolling element's height(or it won't become a scrolling element, right?).
